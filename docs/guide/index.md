# 介绍

欢迎使用 Ares 静态文档站点！

## 什么是 Ares Docs?

Ares Docs 是一个基于 VitePress 构建的现代化文档系统，旨在提供：

- 🚀 **极速体验** - 基于 Vite 的快速热重载
- 🎨 **美观设计** - 简洁优雅的默认主题
- 📱 **响应式布局** - 完美适配各种设备
- 🔍 **内置搜索** - 本地全文搜索支持

## 为什么选择 VitePress?

VitePress 是 Vue.js 生态中的静态站点生成器，具有以下优势：

1. **性能优异** - 基于 Vite 构建，开发和构建速度极快
2. **Vue 驱动** - 可以在 Markdown 中使用 Vue 组件
3. **主题定制** - 灵活的主题系统，易于定制
4. **SEO 友好** - 静态 HTML 生成，利于搜索引擎优化
