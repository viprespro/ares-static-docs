# 游戏设置 - 基础

本章节将介绍游戏的基础设置方法。

## 前置准备

在开始设置之前，请确保：

- ✅ 工具已正确安装
- ✅ 游戏已安装并可正常运行
- ✅ 具备管理员权限（如需要）

## 游戏内设置

### 1. 画面设置

打开游戏，进入设置界面：

| 设置项 | 推荐值 | 说明 |
|--------|--------|------|
| 分辨率 | 1920x1080 | 根据显示器调整 |
| 帧率限制 | 60 FPS | 根据需求调整 |
| 垂直同步 | 开启 | 防止画面撕裂 |

### 2. 控制设置

配置游戏控制选项：

```
设置 -> 控制 -> 自定义按键
```

### 3. 音频设置

调整游戏音频：

```
设置 -> 音频 -> 调整音量
```

## 工具配置

### 连接游戏

1. 启动游戏
2. 运行工具
3. 工具将自动检测游戏进程

::: warning 注意
请确保游戏以管理员权限运行，否则工具可能无法正常工作。
:::

### 基础配置项

在工具中配置以下基础选项：

```json
{
  "game": {
    "autoDetect": true,
    "processName": "game.exe",
    "windowTitle": "游戏名称"
  }
}
```

## 验证设置

完成设置后，请验证：

1. 工具能够正常检测到游戏
2. 各项功能能够正常使用
3. 没有出现错误提示
