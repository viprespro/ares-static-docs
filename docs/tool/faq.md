# 常见问题

本章节汇总了使用过程中常见的问题及解决方案。

## 安装问题

### Q: 安装时提示缺少依赖？

**A:** 请确保已安装所有必需的运行环境。可以尝试：

```bash
# 检查依赖
your-tool check-deps

# 自动安装缺失依赖
your-tool install-deps
```

### Q: Windows 提示"无法运行此应用"？

**A:** 这可能是安全设置问题：

1. 右键点击工具 -> 属性
2. 勾选"解除锁定"
3. 点击确定后重试

或者以管理员身份运行。

## 运行问题

### Q: 工具启动后闪退？

**A:** 可能的原因和解决方案：

1. **缺少运行时** - 安装所需的运行时环境
2. **配置错误** - 删除配置文件重新初始化
3. **权限不足** - 以管理员身份运行

```bash
# 查看错误日志
your-tool --debug
```

### Q: 无法检测到游戏进程？

**A:** 请检查：

1. 游戏是否已启动
2. 游戏是否以管理员权限运行
3. 工具是否有足够的权限

```json
// 手动配置游戏进程名
{
  "game": {
    "processName": "game.exe"
  }
}
```

### Q: 功能无响应或卡顿？

**A:** 尝试以下步骤：

1. 检查 CPU 和内存占用
2. 降低更新频率
3. 关闭不必要的功能

```json
{
  "performance": {
    "updateInterval": 100,
    "maxMemory": 2048
  }
}
```

## 配置问题

### Q: 配置文件在哪里？

**A:** 配置文件默认位置：

```
config/
├── settings.json      # 基础配置
├── advanced.json      # 高级配置
└── hotkeys.json       # 快捷键配置
```

### Q: 如何重置配置？

**A:** 运行以下命令：

```bash
your-tool reset
```

或手动删除配置文件后重启工具。

### Q: 配置修改后不生效？

**A:** 请确保：

1. 配置文件格式正确（有效的 JSON）
2. 已保存配置文件
3. 重启工具使配置生效

## 游戏设置问题

### Q: 游戏设置被重置？

**A:** 部分游戏会自动重置修改的设置：

1. 检查游戏是否有配置保护
2. 将配置文件设为只读
3. 使用工具的自动应用功能

### Q: 如何备份游戏存档？

**A:** 使用工具的备份功能：

```bash
your-tool backup-save --output ./saves/
```

## 其他问题

### Q: 如何更新工具？

**A:** 运行更新命令：

```bash
your-tool update
```

或手动下载最新版本覆盖安装。

### Q: 如何获取帮助？

**A:** 可以通过以下方式：

1. 查看本文档
2. 运行 `your-tool help`
3. 提交 Issue 到 GitHub

### Q: 如何报告 Bug？

**A:** 请提供以下信息：

- 工具版本
- 操作系统版本
- 错误截图或日志
- 复现步骤
