# 激活授权

无极助手 采用卡密授权系统，确保软件的安全性和正版用户的权益。

## 授权方式

### 在线激活（推荐）

需要联网，通过服务器验证激活码：

1. 启动软件后弹出激活窗口
2. 输入您的卡密（激活码）
3. 点击"激活"按钮
4. 等待服务器验证
5. 激活成功后自动进入主界面

### 离线验证

当网络不可用时，软件会自动降级到离线验证模式：

- 使用本地缓存的授权信息
- 离线验证有次数限制（最多10次）
- 恢复网络后会自动重置计数

::: warning 注意
长期离线可能导致授权失效，请定期连接网络验证。
:::

## 授权信息

### 查看授权状态

激活成功后，主界面顶部状态栏会显示：

- **剩余时间**：距离到期还有多久
- **到期时间**：具体的到期日期

### 授权状态颜色

| 颜色 | 状态 | 说明 |
|------|------|------|
| 🟢 绿色 | 已激活 | 授权有效，可正常使用 |
| 🔴 红色 | 已过期 | 授权已到期，需要续费 |
| ⚫ 灰色 | 未激活 | 需要输入卡密激活 |

## 心跳检测

软件会定期（每60秒）向服务器验证授权状态：

- 确保授权仍然有效
- 检测卡密是否被禁用
- 更新授权信息显示

如果卡密被禁用，软件会自动停止运行。

## 更换设备

### 设备绑定说明

激活码与机器码绑定，每个激活码只能在一台设备上使用。

### 如何更换设备

如需更换设备，请联系客服并提供：

1. 原激活码
2. 订单信息
3. 新设备信息（可选）

客服会帮您处理设备转移。

::: tip 提示
建议在更换设备前先联系客服，避免激活码被锁定。
:::

## 续费说明

### 到期提醒

授权即将到期时，软件会提醒您续费。

### 续费方式

联系客服进行续费，续费后时间会累加到当前授权时间。

## 常见问题

### 激活码无效？

可能的原因：

1. **输入错误**：检查大小写和空格
2. **已被使用**：每个激活码只能激活一台设备
3. **已过期**：激活码有时效性

如仍有问题，请联系客服并提供订单信息。

### 网络连接失败？

如果激活时提示网络错误：

1. 检查网络连接是否正常
2. 检查防火墙是否拦截
3. 尝试使用离线验证（如果有本地授权）

### 重装系统后需要重新激活吗？

是的，重装系统会改变机器码，需要重新激活。请联系客服处理。

### 可以同时在多台电脑使用吗？

不可以，一个激活码只能在一台设备上使用。如需多台设备使用，请购买多个激活码。

## 安全说明

### 授权文件

授权信息存储在：

```
%APPDATA%\TLWuJiAssistant\license.lic
```

该文件经过加密，请勿删除或篡改。

### 数据安全

- 所有授权数据本地加密存储
- 不会上传个人隐私信息
- 网络传输使用 HTTPS 加密

::: danger 警告
请勿尝试破解或绕过授权系统，这违反使用协议并可能导致法律问题。
:::
