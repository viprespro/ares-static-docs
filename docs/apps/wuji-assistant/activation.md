# 激活授权

本指南将帮助您快速上手无极助手，从下载安装到激活授权一站式完成。

## 下载应用

### 获取下载链接

购买后会收到下载链接或可运行的exe执行程序，以及对应的卡密（激活码）

::: tip 提示
建议从可信渠道下载，避免使用来路不明的版本
:::

## 安装步骤

### 1. 解压文件

下载完成后，如果是压缩包则将压缩包解压到您希望放置的目录。

::: warning 注意
解压路径**不要包含中文或特殊字符**，建议使用纯英文路径。
:::

### 2. 运行程序

如果是exe执行程序，则双击 `WujiAssistant-vx.x.x.exe` 启动应用即可。

### 3. 首次启动

启动时会显示启动画面，稍等片刻进入激活界面。

## 激活授权

首次使用需要卡密授权，请按照以下步骤操作：

### 授权方式

#### 在线激活

需要联网，通过服务器验证激活码：

1. 启动软件后弹出激活窗口
2. 输入您的卡密（激活码）
3. 点击"激活"按钮
4. 等待服务器验证
5. 激活成功后自动进入主界面

### 授权状态

激活成功后，主界面顶部会显示授权信息：

| 颜色 | 状态 | 说明 |
|------|------|------|
| 🟢 绿色 | 已激活 | 授权有效，可正常使用，显示到期时间 |
| 🔴 红色 | 已过期 | 授权已到期，需要续费 |
| ⚫ 灰色 | 未激活 | 需要输入卡密激活 |

::: warning 注意
- 每个卡密只能在一台设备上使用
- 卡密与机器码绑定，更换设备需联系客服
- 请妥善保管您的卡密，避免泄露给他人
- 如果没看到激活按钮，请调整激活界面窗口高度再试
:::

## 授权信息

### 查看授权状态

激活成功后，主界面顶部状态栏会显示：

- **剩余时间**：距离到期还有多久
- **到期时间**：具体的到期日期

## 更换设备

### 设备绑定说明

激活码与设备绑定，每个激活码只能在一台设备上使用。

### 如何更换设备

如需更换设备，请联系客服并提供：

1. 原激活码

客服会帮您处理解绑原设备，用户自行激活新设备即可。

注：解绑无损耗时间，每个激活码有一次免费的解绑机会，后续解绑2R一次。

::: tip 提示
建议在更换设备前先联系客服，避免激活码被锁定。
:::

## 续费说明

### 到期提醒

授权即将到期时，软件会提醒您续费。

### 续费方式

联系客服进行续费，续费后时间会累加到当前授权时间。

## 常见问题

### 激活码无效？

可能的原因：

1. **输入错误**：检查大小写和空格
2. **已被使用**：每个激活码只能激活一台设备
3. **已过期**：激活码有时效性

如仍有问题，请联系客服并提供订单信息。

### 网络连接失败？

如果激活时提示网络错误：

1. 检查网络连接是否正常
2. 检查防火墙是否拦截
3. 尝试使用离线验证（如果有本地授权）

### 重装系统后需要重新激活吗？

是的，重装系统会改变设备标识，需要重新激活。请联系客服处理。

### 可以同时在多台电脑使用吗？

不可以，一个激活码只能在一台设备上使用。如需多台设备使用，请购买多个激活码。

## 安全说明

### 数据安全

- 所有授权数据本地加密存储
- 不会上传个人隐私信息
- 网络传输使用加密通道

::: danger 警告
请勿尝试破解或绕过授权系统，这违反使用协议并可能导致法律问题。
:::
