# 基础使用

本章节介绍无极助手的基本使用方法。

## 绑定游戏窗口

### 前提条件

1. 已启动天龙八部游戏客户端
2. 游戏窗口已完全加载（进入游戏画面）
3. 游戏窗口可见（不要最小化）

### 绑定步骤

**方式一：**（推荐）
1. 长按主界面的红色"准星图标"不松开
2. 将图标拖动到游戏窗口上
3. 松开鼠标
4. 软件会自动识别并绑定
5. 状态栏显示"已绑定窗口：《天龙八部》 xxx"表示成功

**方式二：**
1. 主界面存在一键绑定按钮，直接点击即可绑定
2. 注意，这种方式适合只有一个游戏窗口的情况，实现快速绑定。


## 挂机设置

### 门派选择

在"挂机设置"选项卡中选择您的角色门派：

- 恶人谷
- 慕容
- 其他门派...

选择门派后，可以根据你的门牌设置对应的技能快捷键。


## 开始挂机

### 启动挂机

配置完成后，点击"开始"按钮开始挂机。

软件会自动：

1. 检测周围的怪物
2. 选择目标并攻击
3. 释放技能
4. 拾取物品（如果启用）

### 暂停/继续

- 点击"暂停"按钮暂停挂机
- 再次点击"继续"按钮恢复挂机
- 或使用快捷键控制（默认 F6）

### 停止挂机

点击"停止"按钮停止挂机，软件会停止所有自动化操作。

## 状态监测

### 死亡检测

启用死亡检测后，软件会：

1. 检测角色是否死亡
2. 根据配置执行相应操作：
   - 自动回城复活
   - 原地复活
   - 什么都不做

### 断线重连

启用断线重连后，软件会：

1. 检测是否断线
2. 自动点击重连按钮
3. 等待重新连接
4. 继续挂机

::: warning 注意
断线重连使用前台鼠标操作，操作期间请勿使用鼠标。
:::

## 快捷键

### 默认快捷键

| 快捷键 | 功能 |
|--------|------|
| F6 | 开始/暂停挂机 |
| F7 | 停止挂机 |
| F8 | 显示/隐藏窗口 |

### 自定义快捷键

在"设置"选项卡中可以自定义快捷键。

## 多开支持

### 多实例限制

软件支持同时运行多个实例（默认最多2个）：

- 每个实例需要绑定不同的游戏窗口
- 每个实例独立运行，互不干扰

::: warning 注意
多开会占用更多系统资源，请根据电脑配置合理使用。
:::

### 多开步骤

1. 启动第一个游戏窗口和助手实例
2. 绑定第一个游戏窗口
3. 启动第二个游戏窗口和助手实例
4. 绑定第二个游戏窗口
5. 分别开始挂机

## 预设配置

### 保存预设

1. 配置好挂机参数
2. 点击"辅助工具" → "保存预设"
3. 输入预设名称
4. 点击保存

### 加载预设

1. 点击"辅助工具" → "加载预设"
2. 选择之前保存的预设
3. 点击加载

### 预设管理

预设文件保存在：

```
%APPDATA%\TLWuJiAssistant\presets\
```

可以备份这些文件到其他位置。

## 注意事项

### 使用建议

1. **首次使用**：建议在安全区域测试，熟悉操作
2. **定期检查**：定期查看挂机状态，避免异常
3. **合理使用**：避免长时间无人值守挂机
4. **保持更新**：及时更新到最新版本

### 避免封号

虽然软件采用后台操作技术，但仍建议：

1. 不要24小时连续挂机
2. 适当休息，模拟正常游戏行为
3. 避免在人多区域挂机
4. 关注游戏官方公告
