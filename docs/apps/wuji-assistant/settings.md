# 设置说明

通过设置中心，您可以根据个人需求定制无极助手的行为。

## 打开设置

点击主界面右上角的"设置"按钮，或使用快捷键 `Ctrl + ,` 打开设置面板。

## 基础设置

### 外观设置

- **主题模式**：选择浅色或深色主题
- **界面语言**：支持中文、英文等
- **字体大小**：调整界面字体大小

### 存储设置

- **数据存储路径**：设置数据文件的保存位置
- **自动备份**：开启/关闭自动备份功能
- **备份间隔**：设置自动备份的时间间隔

### 性能设置

- **启动选项**：设置是否开机自启动
- **内存优化**：调整内存使用策略
- **缓存管理**：清理或管理应用缓存

## 高级设置

### 网络设置

- **代理配置**：如需使用代理，在此配置
- **连接超时**：设置网络请求超时时间

### 日志设置

- **日志级别**：选择记录的日志详细程度
- **日志路径**：设置日志文件保存位置

### 快捷键设置

您可以自定义各个功能的快捷键：

1. 进入"快捷键设置"页面
2. 点击要修改的功能
3. 按下新的快捷键组合
4. 点击"保存"

## 数据管理

### 导出数据

可以将应用数据导出为文件，便于备份或迁移。

### 导入数据

从备份文件中恢复数据。

### 重置设置

将所有设置恢复为默认值。

::: danger 警告
重置设置将清除所有自定义配置，此操作不可撤销！
:::
