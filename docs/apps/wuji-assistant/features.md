# 功能详解

本章节详细介绍无极助手的各项功能。

## 自动打怪

### 怪物识别

软件会自动识别游戏画面中的怪物并进行攻击。

#### 预设怪物

软件内置了常见怪物的识别，开箱即用。

#### 自定义怪物

可以添加自定义的攻击/忽略怪物：

1. 点击"辅助工具" → "怪物管理"
2. 选择"添加攻击怪物"或"添加忽略怪物"
3. 截取怪物特征图片
4. 保存配置

### 攻击模式

- **打死换怪**：击杀当前怪物后自动寻找下一个目标
- **抢怪模式**：快速扫描攻击范围内的怪物

### 攻击方式

- **普通攻击**：使用基础攻击
- **技能攻击**：自动释放技能（需配置技能快捷键）

## 技能调度

### 技能配置

在"挂机设置"中配置技能：

1. 设置技能快捷键（游戏中）
2. 配置技能释放顺序
3. 设置技能冷却时间

### 技能类型

支持多种技能类型：

- **攻击技能**：主要输出技能
- **增益技能**：BUFF类技能
- **回复技能**：回血回蓝技能
- **特殊技能**：门派特色技能

## 状态监测

### 死亡检测

软件会自动检测角色死亡状态，可配置死亡后的行为：

- **自动回城**：自动回城复活
- **原地复活**：使用复活道具原地复活
- **无操作**：不做任何处理，等待人工处理

### 断线重连

启用断线重连后，软件会：

1. 检测到断线
2. 自动点击重连按钮
3. 等待重新连接
4. 继续挂机

::: warning 注意
断线重连期间会短暂使用鼠标，操作期间请勿使用鼠标。
:::

### 窗口尺寸检测

启动时会检测游戏窗口尺寸，支持的尺寸：

- 1024x768
- 1280x720
- 1280x1024
- 1920x1080

## 辅助工具

### 截图工具

可以截取游戏画面：

1. 点击"辅助工具" → "截图"
2. 选择截图区域
3. 保存图片

### 怪物管理

管理自定义怪物配置：

- 添加攻击怪物
- 添加忽略怪物
- 删除怪物配置
- 导入/导出配置

### 预设管理

保存和加载挂机配置：

- 保存当前配置为预设
- 加载已保存的预设
- 删除预设
- 导入/导出预设文件

## 设置选项

### 基础设置

- 窗口位置记忆
- 攻击速度调节
- 门派选择

### 快捷键设置

自定义各功能的快捷键：

- 开始/暂停
- 停止
- 显示/隐藏

### 更新设置

- 启动时自动检查更新
- 定时检查更新
- 手动检查更新

## 数据管理

### 数据备份

可以备份所有配置数据：

1. 点击"设置" → "备份数据"
2. 选择保存位置
3. 生成备份文件

### 数据恢复

从备份文件恢复数据：

1. 点击"设置" → "恢复数据"
2. 选择备份文件
3. 确认恢复

## 多开支持

软件支持同时运行多个实例（默认最多2个）：

- 每个实例需要绑定不同的游戏窗口
- 每个实例独立运行，互不干扰

::: warning 注意
多开会占用更多系统资源，请根据电脑配置合理使用。
:::
