# 无极助手

欢迎使用无极助手！这是一款专为天龙八部端游（怀旧、怀中坏、WG、源端私服等）设计的图色按键类辅助工具，一定程度上帮助您轻松游戏，提升游戏幸福感。

## 简介

无极助手 通过图像识别和自动化技术，实现自动打怪、技能释放、状态监测等功能，让您解放双手，轻松游戏。

## 主要功能

### 🎯 核心功能

- **自动打怪** - 智能识别怪物，支持打死换怪或者抢怪模式，自动攻击
- **打特定怪、玩家** - 截图怪物、玩家图片，实现自动识别打特定怪或玩家
- **技能调度** - 自动释放技能，智能冷却管理
- **状态监测** - 实时检测角色状态（死亡、断线等）
- **断线重连** - 检测到断线后自动重连

### ⚙️ 辅助功能

- **多门派支持** - 支持多个门派的技能配置
- **自定义怪物** - 可添加自定义攻击/忽略怪物
- **预设配置** - 保存和加载挂机配置
- **快捷键控制** - 通过快捷键快速启停

### 🔐 安全保障

- **卡密授权** - 安全的授权验证系统
- **自动更新** - 保持软件最新版本
- **多实例限制** - 防止资源冲突
