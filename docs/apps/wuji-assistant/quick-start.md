# 快速开始

本指南将帮助您快速上手无极助手。

## 下载应用

### 获取下载链接

购买后会收到下载链接或可运行的exe执行程序，以及对应的卡密（激活码）

::: tip 提示
建议从可信渠道下载，避免使用来路不明的版本
:::

## 安装步骤

### 1. 解压文件

下载完成后，如果是压缩包则将压缩包解压到您希望放置的目录。

::: warning 注意
解压路径**不要包含中文或特殊字符**，建议使用纯英文路径。
:::

### 2. 运行程序

如果是exe执行程序，则双击 `WujiAssistant-vx.x.x.exe` 启动应用即可。

### 3. 首次启动

启动时会显示启动画面，稍等片刻即可激活界面。

## 激活授权

首次使用需要卡密授权，请按照以下步骤操作：

### 输入卡密

1. 启动应用后，会弹出激活窗口
2. 输入您收到的卡密（激活码）
3. 点击"激活"按钮
4. 激活成功后自动进入主界面

::: warning 注意
- 每个卡密只能在一台设备上使用
- 卡密与机器码绑定，更换设备需联系客服
- 请妥善保管您的卡密，避免泄露给他人
- 如果没看到激活按钮，请调整激活界面窗口高度再试
:::

### 授权状态

激活成功后，主界面顶部会显示授权信息：

- 🟢 **绿色**：已激活，显示到期时间
- 🔴 **红色**：已过期，需要续费
- ⚫ **灰色**：未激活

## 基础配置

### 1. 游戏设置

1. 非聊模式

### 2. 绑定游戏窗口

1. 启动天龙八部游戏客户端
2. 在无极助手主界面根据提示长按图标拖动到游戏窗口上
3. 软件会自动识别并绑定游戏窗口


### 3. 配置预设

默认会有提供三个预设，分别是：

- **我是峨眉**：代表当前你绑定的窗口时峨眉，会执行峨眉的加血逻辑
- **打死换怪-仅平涂**：代表当前设置为打死换怪，也就是打死怪物后会自动换怪，只平推，不释放技能，可以根据自己的需求自定义
- **抢怪模式-仅平推**：代表当前设置为抢怪模式，也就是扫射，只平推，不释放技能，可以根据自己的需求自定义

### 4. 配置攻击方式

- **普通攻击**：使用基础攻击
- **技能攻击**：自动释放技能

### 5. 开始挂机

点击"开始"按钮，软件将自动开始挂机打怪。
