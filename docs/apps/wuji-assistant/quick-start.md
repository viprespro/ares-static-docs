# 快速开始

本指南将帮助您快速上手无极助手。

## 下载应用

### 获取下载链接

购买后会收到下载链接，通过链接下载最新版本的安装包。

::: tip 提示
建议从官方渠道下载，避免使用来路不明的版本。
:::

## 安装步骤

### 1. 解压文件

下载完成后，将压缩包解压到您希望安装的目录。

::: warning 注意
解压路径**不要包含中文或特殊字符**，建议使用纯英文路径。
:::

### 2. 运行程序

双击 `WujiAssistant-v1.0.7.exe` 启动应用。

### 3. 首次启动

首次启动时会显示启动画面，稍等片刻即可进入主界面。

## 激活授权

首次使用需要激活授权，请按照以下步骤操作：

### 输入激活码

1. 启动应用后，会弹出激活窗口
2. 输入您收到的卡密（激活码）
3. 点击"激活"按钮
4. 激活成功后自动进入主界面

::: warning 注意
- 每个激活码只能在一台设备上使用
- 激活码与机器码绑定，更换设备需联系客服
- 请妥善保管您的激活码
:::

### 授权状态

激活成功后，主界面顶部会显示授权信息：

- 🟢 **绿色**：已激活，显示到期时间
- 🔴 **红色**：已过期，需要续费
- ⚫ **灰色**：未激活

## 基础配置

### 1. 绑定游戏窗口

1. 启动天龙八部游戏客户端
2. 在无极助手中点击"绑定窗口"按钮
3. 将鼠标移动到游戏窗口上
4. 软件会自动识别并绑定游戏窗口

### 2. 选择门派

在"挂机设置"中选择您的角色门派，软件会自动配置对应的技能。

### 3. 配置攻击方式

- **普通攻击**：使用基础攻击
- **技能攻击**：自动释放技能

### 4. 开始挂机

点击"开始"按钮，软件将自动开始挂机打怪。
